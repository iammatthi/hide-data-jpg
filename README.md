## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

- python 3
- pip3

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/iammatthi/hide-data-jpg
   ```
2. Install python packages
   ```sh
   pip3 install -r requirements.txt
   ```
